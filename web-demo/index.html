EV Adoption Forecast System

Problem Statement

Electric Vehicle (EV) adoption is increasing rapidly, but predicting future growth is complex due to economic and technological factors. This project aims to forecast EV adoption (2010–2030) using mathematical models: Logistic, Gompertz, and Hybrid.

Dataset

EV Sales

Year-wise Analysis

2010–2018 → Slow adoption
2019–2021 → Initial growth
2021–2024 → Rapid boom
2025–2030 → Expected stabilization

Models & Formulas

Logistic Model

K / (1 + e^(-r(t - t0)))

Gompertz Model

K * exp(-exp(-r(t - t0)))

Hybrid Model

0.4 × Logistic + 0.6 × Gompertz
